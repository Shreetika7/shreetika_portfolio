Learning & Development Report

A reflection on my journey building this portfolio.

Overview

My Learning Journey

Over this term, I learned web development through online resources. My interest in coding and web design grew as I explored HTML, layouts, and design principles. Building this portfolio allowed me to practice creating a clean, responsive, and aesthetically pleasing website.

"I faced challenges like maintaining consistent design across pages, structuring the layout, and making the site responsive for mobile devices."

Experimenting with inline styles, grid layouts, and hover effects helped me understand practical web design more deeply.

Design Decisions

Choices & Rationale

Design decisions included choosing a pink-and-white theme for an approachable and elegant look. Cormorant Garamond was selected for headings for a sophisticated, editorial feel, and Jost was used for body text to enhance readability with a clean, modern style.

The site includes a homepage introducing me, a projects page showcasing skills with a grid layout, a video demo embedded via iframe, a contact page with a mailto form, and this report documenting the learning process.

Reflection

What I Gained

This project strengthened my understanding of HTML, design principles, layout structuring, and responsive techniques. I learned to balance aesthetics and functionality while keeping the code simple and maintainable.
