Learning & Development Report

Over this term, I learned web development through online resources. My interest in coding and web design grew as I explored HTML, layouts, and design principles. Building this portfolio allowed me to practice creating a clean, responsive, and aesthetically pleasing website. I faced challenges like maintaining consistent design across pages, structuring the layout, and making the site responsive for mobile devices. Experimenting with inline styles, grid layouts, and hover effects helped me understand practical web design better.

Design decisions included choosing a pink-and-white theme for an approachable and elegant look, Playfair Display for headings for sophistication, and DM Sans for body text to enhance readability. The site includes a homepage introducing me, a projects page showcasing skills with a grid layout, a video demo embedded via iframe, a contact page with a mailto form, and this report documenting the learning process. Validation involved checking responsiveness, link functionality, and cross-page consistency.

This project strengthened my understanding of HTML, design principles, layout structuring, and responsive techniques. I learned to balance aesthetics and functionality while keeping the code simple. It boosted my confidence as a beginner web developer. Watch my video demonstration here: Video Demo.
